This project was developed as an assignment for INFO 474: Interactive Information Visualization at the University of Washington Information School.

The visualization shows the brightness and fire radiative power of wildfires in Australia over time as captured by NASA satellites. The satellite, time of day, and data representation can be adjusted to explore more deeply.

Wildfire dataset from Kaggle user Carlos Paradis /
Australia geojson from GitHub user tonywr71
